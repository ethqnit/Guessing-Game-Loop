# Guessing Game Loop – With Epic ASCII Art!

**A classic Java number guessing game** with a twist: every wrong guess triggers **insane ASCII art** — one for "too high", one for "too low"!

Made in the golden era of console apps. Works perfectly in Eclipse, NetBeans, or terminal.

## Features

- Random number between 1–100
- Unlimited guesses
- Two **hand-crafted ASCII masterpieces** on every wrong guess
- Counts your attempts
- Pure Java — no dependencies
- Works 100% in 2025

<br>

## How to Run

1. Save as `GuessingGame.java`
2. Compile & run:
```bash
javac GuessingGame.java
java GuessingGame
Or just open in Eclipse / NetBeans / IntelliJ and click Run
